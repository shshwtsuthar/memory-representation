# AgentInstruct Dataset

## Description

AgentInstruct is a lightweight instruction-tuning dataset containing high-quality interaction trajectories designed to enhance the agent capabilities of Large Language Models (LLMs). The dataset was created as part of the AgentTuning methodology, which aims to improve LLMs' ability to act as agents in complex real-world tasks while maintaining their general language capabilities.

The data includes subsets of alfworld, db, kg, mind2web, os, webshop. This folder includes the subset of os.
