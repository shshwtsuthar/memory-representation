# Code-Feedback Dataset

## Description

Code-Feedback is a comprehensive dataset featuring 66.4K multi-turn interactions designed for training code generation models with execution capabilities and iterative refinement. The dataset was created to support the development of OpenCodeInterpreter, an open-source code system that integrates code generation, execution, and human feedback for dynamic code refinement.

Code-Feedback enables models to bridge the gap between open-source code generation systems and proprietary systems like GPT-4 Code Interpreter by providing rich training data that includes both successful code generation and the iterative refinement process based on execution results and feedback.
