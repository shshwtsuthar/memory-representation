# Orca Agentinstruct Dataset

## Description

Orca Agentinstruct is a large-scale synthetic dataset containing tool-use instruction examples created using the AgentInstruct framework. The dataset was designed for post-training language models to teach various skills including tool usage, text editing, creative writing, coding, and reading comprehension.

The dataset focuses on:
- Large-scale synthetic tool-use instructions data
- Diverse and high-quality synthetic data generation
- Multi-skill training including text editing, creative writing, tool usage, coding, and reading comprehension
- Automated creation of both prompts and responses using raw data sources

AgentInstruct is an extensible agentic framework that automatically creates large amounts of diverse and high-quality synthetic data using only raw data sources like text documents and code files as seeds.
