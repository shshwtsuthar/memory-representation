# Mini-Coder Dataset

## Description

Mini-Coder is a large-scale dataset of verified, multi-turn coding trajectories designed to train compact software engineering agents. The trajectories are generated using the mini-swe-agent workflow on SWE-smith tasks with Qwen 3 Coder 30B A3B as the backbone model.

This dataset focuses on:
- Scalable supervision for agentic software engineering
- Multi-turn, tool-using behavior in a deterministic shell interface
- Verified trajectories from PR-style bug-fix tasks sourced from SWE-smith

Used to train the Mini-Coder models.
