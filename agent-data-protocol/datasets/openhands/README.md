# openhands-feedback Dataset

## Description

openhands-feedback is a dataset containing recorded OpenHands agent trajectories with human feedback. The dataset captures real agent interactions across multiple domains including coding, tool use, and web browsing, enhanced with human feedback to improve agent performance.

The dataset focuses on:
- Programming and code generation tasks
- External tool and API interaction scenarios
- Web navigation and interaction tasks
- Recorded agent trajectories with human feedback integration

The dataset provides valuable training data that combines automated agent behavior with human guidance, enabling the development of more effective and aligned AI agents. The trajectories capture real-world usage patterns of the OpenHands agent system across diverse task domains.
