# Synatra Dataset

## Description

Synatra is a dataset containing synthetically created web demonstrations generated by transforming indirect knowledge (such as online tutorials) into direct supervision for training web agents. The dataset addresses the challenge of obtaining large-scale, direct demonstrations for digital tasks by leveraging abundant indirect knowledge sources.

The dataset focuses on:
- Synthetically created web demos of tutorials
- Transformation of indirect knowledge into direct demonstrations
- Large-scale synthetic data generation for web-based tasks
- Cost-effective alternative to human demonstrations
