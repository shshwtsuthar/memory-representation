# SWE-Gym Dataset

## Description

SWE-Gym is the first environment for training real-world software engineering (SWE) agents, containing agent trajectories solving real GitHub repository tasks. The dataset was created from real-world Python task instances, each comprising a codebase with an executable runtime environment, unit tests, and a task specified in natural language.

The dataset focuses on:
- Agent trajectories solving real GitHub repo tasks
- Real-world Python task instances from GitHub issues
- Executable runtime environments with unit tests
- Natural language task specifications
