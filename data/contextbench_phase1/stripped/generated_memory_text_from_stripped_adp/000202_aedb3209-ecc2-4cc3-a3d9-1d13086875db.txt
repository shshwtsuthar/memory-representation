```

                   for i in range(1, self.rows)
                   for j in range(i))
```

```
>>> A = sympy.zeros(4, 2)
>>> print tuple([i, j] for i in range(1, A.rows) for j in range(i))

```
The attempt to index the (3,2) entry appears to be the source of the error.
