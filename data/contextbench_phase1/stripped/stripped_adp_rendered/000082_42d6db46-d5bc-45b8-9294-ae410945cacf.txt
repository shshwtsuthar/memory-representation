The `flask routes` command is implemented as a Click command in the Flask CLI module. It was originally added in [PR #2259](https://github.com/pallets/flask/pull/2259) by davidism (merged April 26, 2017).

### 2. Current Route Display Format

**Original Implementation (Flask < 2.3.0):**
The command displays routes in a tabular format with three columns:
- **Endpoint**: The name of the route endpoint
- **Methods**: HTTP methods (GET, POST, PUT, DELETE, etc.) with HEAD and OPTIONS ignored
- **Rule**: The URL pattern/route path

**Enhanced Implementation (Flask 2.3.0+):**
Following [Issue #5004](https://github.com/pallets/flask/issues/5004), the command was enhanced to conditionally display subdomain/host information. The commit ([84c007d](https://github.com/pallets/flask/commit/84c007d34f7f0b8737310fe4d67be12c463e0130)) added:
- Dynamic column detection for **Subdomain** or **Host** (when applicable)
- Domain-based sorting option
- Improved formatting logic
